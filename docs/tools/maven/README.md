# Apache Maven

## Introduction

- https://maven.apache.org/

Apache Maven（あぱっち めいゔん／めいゔぇん）は、Java 用プロジェクト管理ツールであり、Apache Ant に代わるものとして作られた、Apache ライセンスにて配布されているオープンソースソフトウェアです。
